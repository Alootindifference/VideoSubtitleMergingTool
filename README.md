# VideoSubtitleMergingTool_V1.0
视频字幕合并工具v1.0正式上线支持单视频和字幕合并和批量视频字幕合并
支持视频格式为：mp4，mkv，avi
格式字幕支持：ass，srt，vtt格式
需要pc安装ffmpeg才能使用
支持中文
ffmpeg下载地址：https://ffmpeg.org/
